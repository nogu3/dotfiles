# CLAUDE.md (user-level)

全プロジェクト共通の Claude Code 作業方針。

## Git コミット

### コミット対象はセッション中に自分が編集したファイルのみ

`git commit` を作成する際、当該セッション中に Claude 自身が編集したファイルのみを `git add` する。セッション開始時点で既に `modified` だったファイル（ユーザーが事前に編集した未コミット変更）は、明示的に指示されない限りコミットに含めない。

**理由:** ユーザー作業途中の未完成変更を巻き込むと、意図しないコミットが発生する。セッション中の編集範囲とユーザーの既存作業範囲を混ぜない。

**運用:** セッション開始時の `git status` を記憶し、今回編集したファイルと事前変更ファイルを区別する。事前変更ファイルを扱うかどうかは明示確認する。
